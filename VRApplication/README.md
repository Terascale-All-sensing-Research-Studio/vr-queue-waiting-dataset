# VR Application

A single zip file called [reception.zip](reception.zip) is provided that contains the entire VR application used in this work. The application was created using Unity 2022.3.9.f1. Once downloaded, the application can be imported into a new empty Unity project. Should you run into any issues, please contact us at the [Terascale All-sensing Research Studio](https://tars-home.github.io).
